# Nimbus_Project_YogeshSharma_55
Food delivery order and tracking system
Overview-

This project is a C-based simulation of a food delivery management system similar to apps like Swiggy or Zomato. It allows you to add new food orders, update their delivery status, and generate basic performance reports.

Feature-

1.Add New Orders:
Create new orders with details such as order ID, customer name, restaurant name, status, and delivery time.

2.Update Order Status:
Change the status of any order (e.g., Placed, Preparing, Out for Delivery, Delivered).

3.View Delivery History:
Display all stored orders and their details using loops and arrays of structures.

4.Performance Report:
Generate summaries like average delivery time and number of completed orders.

5.Dynamic Handling with Pointers:
Orders are managed using pointers for efficient data access and updates.
